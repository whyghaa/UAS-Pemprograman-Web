UAS PEMPROGRAMAN WEB

Jelaskan cara kerja Website!
Jawab :
Cara kerja website dimulai dari situs menyimpan informasi melalui dokumen yang berbentuk halaman atau webpage. Webpage tersebut akan disimpan dalam server web dan user sebagai unsur penting pada cara kerja website akan bergerak sebagai komputer client. Ketika pengguna membuka browser dan mengetikkan alamat URL, browser kemudian meminta halaman web dari server web. Setelah server web menerima permintaan dari pengguna, server web akan memproses permintaan tersebut dan menghasilkan respons yang kemudian akan dikirimkan kembali ke browser. Setelah itu, browser akan menampilkan halaman web yang diminta oleh pengguna.
Jelaskan hubungan antara Web Server, Domain dan Hosting!
Jawab :
Dalam proses hosting situs web, web server, domain, dan hosting hubungan dari ketiga tersebut bekerja sama untuk menyediakan konten situs web yang dapat diakses oleh pengguna melalui alamat web. Web Server menyimpan dan menyajikan konten situs web, Domain menyediakan alamat web yang dapat diakses oleh pengguna, dan Hosting menyediakan ruangan server dan kebijakan yang diperlukan untuk menyimpan dan menyajikan konten situs web. Jika dijelaskan :
1. Web server adalah perangkat lunak yang menyimpan dan menyajikan konten (seperti halaman web, gambar, video, dan teks) yang dibutuhkan oleh pengguna untuk mengakses situs web.
2. Domain adalah alamat web yang digunakan oleh pengguna untuk mengakses konten situs web. Domain host menyimpan nama domain dan memfasilitasi registrasi nya.
3. Hosting adalah layanan yang menyediakan ruangan server dan kebijakan yang diperlukan untuk menyimpan dan menyajikan konten situs web.
